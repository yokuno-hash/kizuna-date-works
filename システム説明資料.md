# 作業支援システム 説明資料

合同会社 絆 / 絆データワークス　｜　2026年4月

---

## 1. システム概要

レシートのOCR入力練習を行うためのウェブシステムです。
AIが毎回異なるレシート画像を自動生成し、スタッフが品目・金額を入力する練習を繰り返します。
ブラウザのURLを開くだけで利用でき、アプリのインストールは不要です。

**解決できること**

- 紙のレシートを用意する手間・コストが不要
- 毎回違うレシートが出るため飽きずに練習できる
- 管理者が各スタッフの進捗を数字で把握できる
- 在宅勤務中でも同じ環境でトレーニングが可能

---

## 2. 利用者の使い方

### ログイン
- 管理者から配布されたログインIDとパスワードを入力してログインします

### タスクの流れ

1. ログインすると自動的にレシート画像が表示される
2. 画面左にAI生成のレシート画像（感熱紙風）が表示される
3. 画面右の入力欄に品目名と金額を1行ずつ入力する
4. 「送信する」ボタンを押すと即座に採点される
5. 正解メッセージが表示され、進捗カウントが1増える
6. 「次のタスクへ」ボタンを押すと次のレシートが表示される
7. これを繰り返して月間ノルマを達成する

### 利用者画面の機能

| 機能 | 内容 |
|------|------|
| 進捗バー | ヘッダーに今月の達成枚数 / ノルマ枚数を常時表示（例：125 / 800） |
| AIレシート画像 | 毎回異なる店舗・品目・金額のレシートが自動生成される |
| 品目入力テーブル | 品目ごとに1行ずつ入力。「＋ 行を追加」で行数を増やせる |
| 小計・消費税・合計の自動計算 | 金額を入力するたびに自動で計算・表示される。入力不要 |
| 即時採点 | 「送信する」を押すと結果がすぐ表示される |
| スキップ | 画像が表示されない場合に次の問題へ進める |
| 月間上限 | 月のノルマ（デフォルト800枚）に達すると翌月まで新タスクは出ない |

## 3. 管理者の使い方

管理者専用URLにアクセスすると管理者画面が表示されます。
画面上部のタブで各機能を切り替えます。

---

### タブ① ユーザー管理

スタッフのアカウントを管理します。

- **利用者の追加**：表示名・ログインID・パスワードを入力して追加
- **利用者の削除**：不要になったアカウントを削除
- **パスワードリセット**：パスワードを忘れたスタッフのパスワードを変更できる
- **管理者アカウントの作成**：管理者権限を持つアカウントも同画面から作成できる

---

### タブ② タスク管理

登録済みのタスクを確認・管理します。

- 登録済みタスクをカード形式で一覧表示
- 手動で新しいタスクを追加できる（画像URLと正解テキストを指定）
- タスクを個別に削除できる
- **全件削除ボタン**：登録済みのタスクをまとめて削除できる
- ※ 現在は利用者が「次へ」を押すたびにAIがオンデマンドで生成するため、事前にタスクを登録しておく必要はない

---

### タブ③ 回答管理

全スタッフの回答履歴を確認します。

- ユーザー名・回答日時・カテゴリ・回答内容・正解内容・正誤を一覧表示
- **ユーザー絞り込み**：特定スタッフの回答だけを表示できる
- **レシートサムネイル**：回答したレシートの画像サムネイルが表示される。クリックで拡大表示
- 回答内容と正解内容を並べて確認できるため、入力ミスの傾向を把握しやすい
- 「最新に更新」ボタンでリアルタイムにデータを取得できる

---

### タブ④ 進捗管理

全スタッフの今月の達成状況を確認・管理します。

- 全員の達成枚数・達成率を一覧表示
- **ノルマ変更**：月間ノルマ枚数を変更できる。変更は全員に即時反映（デフォルト：800枚/月）
- **CSVダウンロード**：進捗データをCSV形式でダウンロードできる（Excelで開いて加工も可）
- **月次更新ボタン**：先月分のデータを整理し今月分をリセットする（手動実行）
- **自動実行トリガー設定**：毎月1日0時に自動で月次更新を実行するよう設定できる。設定後は毎月の手動操作が不要

## 6. システム構成

| コンポーネント | 使用サービス | 役割 |
|---|---|---|
| アプリ本体 | Google Apps Script | 画面配信・ログイン・採点・進捗管理 |
| データベース | Google スプレッドシート | ユーザー・回答・進捗データの保存 |
| AI生成 | Vercel + Google Gemini API | レシートテキストの自動生成 |
| 画像生成 | ブラウザ内 Canvas | レシート画像をリアルタイムで描画 |

サーバーの保守・管理は不要です。すべてクラウドで動作します。

**データの安全性**

- パスワードは暗号化（SHA-256ハッシュ）して保存。平文では保管しない
- AIへ送信するのはレシートの架空データのみ。個人情報は一切送信しない
- データはすべて管理者のGoogleアカウントに紐づいたスプレッドシートに保存される

## 7. よくある質問

**Q. スタッフのパスワードを忘れた場合は？**
管理者画面の「ユーザー管理」タブからパスワードをリセットできます。

**Q. ノルマ枚数を途中で変更したい**
「進捗管理」タブの「月間ノルマ設定」から変更できます。月の途中での変更も可能で、全員に即時反映されます。

**Q. 月が変わったら何か操作が必要？**
毎月1日0時に自動実行するよう設定できます（自動実行トリガーの設定）。設定済みであれば毎月の手動操作は不要です。

**Q. 回答内容は後から確認できる？**
「回答管理」タブで全員の回答履歴を確認できます。ユーザーで絞り込むと特定スタッフの回答のみ表示できます。

**Q. 利用者が増えた場合は？**
「ユーザー管理」タブから随時追加できます。利用者数の上限はありません。

**Q. スマートフォンでも使える？**
ブラウザがあれば動作しますが、画面が2カラムレイアウトのためPCでの使用を推奨します。
