# GAS版 作業支援システム セットアップ手順

## ファイル構成

```
gas/
├── Code.gs        ← バックエンドロジック（GASスクリプトエディタに貼り付け）
├── UserPage.html  ← 利用者向けHTML（GASスクリプトエディタに貼り付け）
├── AdminPage.html ← 管理者向けHTML（GASスクリプトエディタに貼り付け）
└── README.md      ← この手順書
```

---

## ステップ1: Googleスプレッドシートを用意する

1. Google ドライブ (drive.google.com) を開く
2. 「新規」→「Googleスプレッドシート」で新しいスプレッドシートを作る
3. URLの `https://docs.google.com/spreadsheets/d/【ここ】/edit` の部分をコピーしておく

---

## ステップ2: GASプロジェクトを作成する

1. スプレッドシートのメニューから「拡張機能」→「Apps Script」を開く
2. 左メニューの「ファイル」横の「＋」から以下3ファイルを追加する：
   - `Code.gs`（最初から存在）
   - `UserPage.html`（HTMLファイルとして追加）
   - `AdminPage.html`（HTMLファイルとして追加）
3. それぞれのファイルに、対応するコードの内容を コピー＆ペースト する

---

## ステップ3: スプレッドシートIDを設定する

`Code.gs` の一番上にある以下の行を書き換える：

```javascript
// 変更前
const SPREADSHEET_ID = 'ここにスプレッドシートのIDを貼り付ける';

// 変更後（例）
const SPREADSHEET_ID = '1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms';
```

---

## ステップ4: シートの初期セットアップを実行する

1. スクリプトエディタの上部の関数選択プルダウンから **`setupSheets`** を選ぶ
2. 「▶ 実行」ボタンをクリック
3. 初回はGoogleアカウントの権限許可を求められるので「続行」→「許可」する
4. 実行ログに `✅ セットアップ完了！` が表示されればOK

> これでスプレッドシートに4つのシート（users / tasks / answers / progress）が自動で作成され、
> 管理者アカウント（ID: `admin` / PW: `admin1234`）も作成されます。

---

## ステップ5: Webアプリとしてデプロイする

1. スクリプトエディタ右上の「**デプロイ**」→「**新しいデプロイ**」をクリック
2. 種類の選択で「**Webアプリ**」を選ぶ
3. 以下のように設定する：

   | 項目 | 設定値 |
   |------|--------|
   | 説明 | 任意（例：作業支援システム v1） |
   | 次のユーザーとして実行 | **自分** |
   | アクセスできるユーザー | **全員** |

4. 「デプロイ」ボタンをクリック
5. 表示される **WebアプリのURL** をコピーする

---

## ステップ6: 動作確認

| 画面 | URL |
|------|-----|
| 利用者画面 | WebアプリのURL（そのまま） |
| 管理者画面 | WebアプリのURL + `?page=admin` |

### 初回ログイン（管理者）

- 管理者画面URLにアクセス
- ID: `admin` / PW: `admin1234` でログイン
- ⚠️ **ログイン後すぐにパスワードを変更してください**

---

## タスクの画像について

タスクの画像はGoogleドライブで管理します：

1. GoogleドライブにPNG/JPEGファイルをアップロード
2. ファイルを右クリック →「共有」→「リンクを知っている全員が閲覧可」に変更
3. 「リンクをコピー」する
4. コピーしたリンクを以下の形式に変換して使う：

```
元のリンク:
https://drive.google.com/file/d/【ファイルID】/view?usp=sharing

画像URL（タスクに登録する値）:
https://drive.google.com/uc?export=view&id=【ファイルID】
```

---

## コードを更新した場合

ソースコードを変更したあとは、必ず「デプロイ」→「**デプロイを管理**」→「✏️ 編集」→「バージョン: 新しいバージョン」→「デプロイ」が必要です。
